Проект можно запускать без подключение к локальному серверу базы данных, так как она была перенесена на хост.

База данных создавалась на MSSQL2017. Для подключение базы данных необходимо иметь MSSQL тойже версии либо выше. 
Чтобы поключить Базу данных необходимо в MSSQL SMS произвести пару действий. 
А именно ПКМ нажать на папку "Databases" -> "Attach...". В открывшемся окне нажать кнопку "Add..." и показать расположение mdf файла.
